# 帕帕韦斯特雷共和国 · 官网

一个虚拟岛国的品牌形象网站，纯属虚构，但做得也像假的一样。

👉 [在线预览](https://papawestray.github.io/)，但现在出了点问题，不知道怎么修，但正在修了。

---

## 这是什么？

我们虚构了一个叫“帕帕韦斯特雷共和国”的国家，顺手给它做了个正儿八经的官方网站。

页面里塞了国家概况、国旗国徽、政府架构、新闻动态、旅游介绍，还有一个能假装提交的联系表单。设计上想要一种**端庄但不死板**的感觉，所以用了深蓝+金色的配色，加上衬线字体，尽量模仿真实国家官网的气质。

所有代码都在一个 `index.html` 文件里，没有用任何框架或构建工具，直接浏览器打开就能看。很适合扔到 GitHub Pages 上，当个虚拟国家的门面。

## 怎么用？

### 部署到 GitHub Pages（推荐）

1. Fork 或上传这个项目到你的 GitHub 仓库
2. 确保 `index.html` 在根目录
3. 进 Settings → Pages，Source 选 `main` 分支，点保存
4. 几分钟后访问 `https://papawestray.github.io` 就能看到了

### 本地预览

因为就一个 HTML 文件，直接双击打开 `index.html` 就行。  
如果用 VS Code，装个 Live Server 插件体验会更好（避免本地加载字体的权限问题）。

## 想改点内容？

整个网站的内容和样式都写在一起，修改起来很直观：

- **改国家名字**：全局搜索“帕帕韦斯特雷”替换掉
- **改配色**：打开文件找到最上面的 `:root` 样式，调 `--color-primary` 和 `--color-gold` 的色值
- **改新闻/景点**：在对应的 `<section>` 里改文字就行
- **加新页面**：复制一个 `<section>` 块，然后在导航栏加个链接
- **换图标**：目前用的都是 emoji 和内联 SVG，换成你自己的图片也完全可以

表单是前端模拟的，不会真的发邮件。如果你想连个后端，把表单的 `submit` 事件换成 `fetch` 请求就行。
